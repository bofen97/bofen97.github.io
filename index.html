How to Use This Helm Chart

To use this Helm chart, follow these steps:

  1. Add the repository to your Helm installation: 
    helm repo add myrepo https://bofen97.github.io/helm-repo
  2. Update your repositories: 
    helm repo update
  3. Install the chart: 
    helm install my-release myrepo/pineapple

Replace 'my-release' with your desired release name and 'pineapple' with the actual chart name from the index.yaml file.
